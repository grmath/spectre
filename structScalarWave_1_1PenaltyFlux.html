Detailed Description

template<size_t Dim>
struct ScalarWave::PenaltyFlux< Dim >

Computes the penalty flux for the ScalarWave system.

The penalty flux is given by:

\begin{align} G(\Psi) &= F(\Psi) = 0 \\ G(\Pi) &= (n_i F^i(\Pi))_{\mathrm{int}} + \frac{1}{2} p \left(v^{-}_{\mathrm{int}} - v^{+}_{\mathrm{ext}}\right) \\ G(\Phi_i) &= (n_i F(\Phi_i))_{\mathrm{int}} - \frac{1}{2} p \left( (n_i v^{-})_{\mathrm{int}} + (n_i v^{+})_{\mathrm{ext}}\right) \end{align}

where \(G\) is the interface normal dotted with numerical flux, the first terms on the RHS for \(G(\Pi)\) and \(G(\Phi_i)\) are the fluxes dotted with the interface normal (computed in ScalarWave::ComputeNormalDotFluxes), and \(v^{\pm}\) are outgoing and incoming characteristic fields of the system (see characteristic_fields() for their definition). The penalty factor is chosen to be \(p=1\).
